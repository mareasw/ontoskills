# Office Suite Skill

This is a declarative meta-skill that groups together document manipulation tools.
It resolves intents related to office automation, document creation, and spreadsheet management.
It does not execute code directly, but serves as a category for its sub-skills:
- docx (Word documents)
- xlsx (Excel spreadsheets)
- pptx (PowerPoint presentations)
- pdf (PDF documents)

Dependencies: None directly, requires the sub-skills to function.
